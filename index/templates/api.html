API

dopepod employs a simple, easy-to-use API for searching and looking up podcasts.

Here's how it works:

To return results in JavaScript Object Notation (JSON) format, use the URL https://dopepod.me/api/json/.

Likewise, to return results in XML format, use the URL https://dopepod.me/api/xml/. (Coming soon!)

By default, results for the top 50 podcasts will be returned.

Search:

To search for podcasts, you may use one or more of the following query parameters:

  • "q", literally querystring, will filter podcasts by title/artist. Example: https://dopepod.me/api/json/?q=dope.
  • "genre" will filter podcasts by genre. Example: https://dopepod.me/api/json/?genre=Arts. For a full list of genres, click here. (Coming soon!)
  • "language" will filter podcasts by language. Example: https://dopepod.me/api/json/?language=English. For a full list of languages, click here. (Coming soon!)
  • "order" will set the order for shown results. Choices: "rank" (default) for ordering based on popularity, or "name" for alphabetical ordering. Example: https://dopepod.me/api/json/?order=name.
  • "show" will set the number of shown results in the range of 1-200. Default is 50. Example: https://dopepod.me/api/json/?show=10.

Note: all spaces must be replaced by a plus sign (+) and ampersands with the URL symbol "%26". Example: https://dopepod.me/api/json/?genre=Sports+%26+Recreation.

Query parameters consist of a key/value pair concatenated with an equal sign (=), such as "q=something". Parameters may be combined to make more complex search queries. First parameter must be preceded by a question mark (?) and each following parameter must be concatenated with an ampersand (&). Order of parameters is not important. Example: https://dopepod.me/api/json/?order=name&genre=Comedy&q=d&show=30&language=German.

Lookup:

To lookup data on a single podcast, you may use the query parameter "podid" in conjuction with the id of the podcast. Podcast id can be found either in the API search results (as "podid"), or on the URL of the podcast, such as https://dopepod.me/showpod/730254722/, where the id is "730254722". So to use API lookup for the aforementioned podcast, you would use the URL https://dopepod.me/api/json/?podid=730254722. (Coming soon!). Note: parameter "podid" will override all other parameters.
